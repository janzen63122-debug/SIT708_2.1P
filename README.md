# Pass Task 2.1 - Travel Companion App

## About the task
This task is about making a Travel Companion App in an Android smartphone, learning how to manage Android Studio & its foundamental function

## Features
In this app you can have simple coversion of:\
**Currency:** USD to AUD, EUR, JPY, and GBP.\
**Fuel & Volume:** MPG to km/L ,US Gallons & Liters.\
**Distance:** Nautical Miles to Kilometers.\
**Temperature:** Celsius to Fahrenheit and Kelvin.\
**Validation:** Error message on same unit conversion, different logic unit conversion (eg. USD to fuel), non-numeric number, and negative numbers on fuel.
  
## How to Use
1. copy whole project to in your computer.
2. Open the project in Android Studio.
3. Run the app on an Android Emulator or a physical Android device
